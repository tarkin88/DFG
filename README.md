# Dot Files Generator
A repository for backup my dotfiles. 

* i3 gaps
* polybar
* rofi
* vim
* rxvt-unicode
* ranger
* mpd 
* ncmpcpp
* dunst 
* compton 
* i3lock-color
* arc theme (for magoon)
* magoon theme
* Font Awesome, Monaco and Source Sans

## how to?
if is clean install:
```bash
chmod +x first_install.sh

./first_install.sh
```
Then you can use the generate script 

```bash
chmod +x generate.sh

./generate.sh
```
